HTML5 Form Demo

New HTML5 form types with Bootstrap.

Number

<input type="number"> <%= f.number_field :attr %>

Number (with pattern=)

<input type="number" pattern="[0-9]*"> <%= f.number_field :attr, pattern: "[0-9]*" %>

Email

<input type="email"> <%= f.email_field :attr %>

HTML5 input types give us improved:

  • Validation (sometimes)
  • Input Affordance (sometimes)

¯\_(ツ)_/¯

Right now, this is very helpful for a very small percent of users (depending on your userbase)

Currency

$ /hour
<input type="number" step="0.01"> <%= f.number_field :attr, step: 0.01, prepend: '$', append: '/hour' %>

Dates/Times

<input type="date"> <%= f.date_field :attr %>

Submits as yyyy-mm-dd

<input type="time"> <%= f.time_field :attr %>

Submits as 23:55

<input type="datetime-local"> <%= f.datetime_local_field :attr %>

Placeholder

<input type="email" placeholder="me@example.com"> <%= f.email_field :attr, placeholder: "me@example.com" %>
